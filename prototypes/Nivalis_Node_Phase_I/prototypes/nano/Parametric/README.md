# QESR Layer 3 – Parametric Visualization

This directory contains the parametric function description for QESR Layer 3: the Nodal Inversion Junction, part of the Halessa Fusion Nivalis Node System.

## Files

- `QESR_L3_parametric.md`: Markdown-ready documentation of the surface equations and their dynamics.
- `QESR_L3_parametric_equations.txt`: Clean equations suitable for pasting into Hyperspace renderer.
- `README.md`: This file.

## Visualization Settings (Suggested)
- u range: 0 to 2π  
- v range: −π to π  
- Resolution: 128 x 128  
- Surface: wireframe + shaded, opacity ~0.8  
- Dynamic modulation: enable vibration along z-axis via `sin(3u)` amplitude
